# Filmbase Product Spec

## Core Value
Movie poster collection and trailer watching.

## MVP Features
- User login
- Movie library
- Poster view
- List view
- Movie detail page
- Trailer playback
- Add movie by search or IMDb URL
- Edit trailer link
- Custom poster upload
- Supabase sync

## Not MVP
- Chat
- Marketplace
- Android
- IMCDb
- Poster designer database
- Real-time IMDb / RT updates
